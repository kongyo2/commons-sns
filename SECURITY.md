# セキュリティポリシー

Commonsは現在プロトタイプ段階のプロジェクトです。脆弱性を発見された場合は、悪用を防ぐため公開Issueには書かず、以下のいずれかの方法で非公開に報告してください。

## 報告方法

- 本リポジトリの **Security** タブ → **Report a vulnerability**（Private vulnerability reporting）
- メール: **anitigravity.lab@gmail.com**

再現手順、影響範囲、可能であれば検証用のコードやログを含めていただけると助かります。

## 対応の目安

プロトタイプ段階で対応体制が小さいため、確約はできませんが、目安として以下を心がけます。

- 報告受領から**1週間以内**に確認の連絡をします
- 修正の要否と対応方針は、確認後にあらためてご連絡します

## 公開について

GOVERNANCEに定めるとおり、セキュリティ対応は解決するまで非公開で進めます。修正を公開した後、必要な範囲で経緯を公開し、ご希望があれば報告者のお名前をクレジットします。
